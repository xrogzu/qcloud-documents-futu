DBAccount

云数据库账号信息

名称 类型 是否必选 描述
UserName String 是 用户名
Host String 是 用户可以从哪台主机登录（对应 MySQL 用户的 host 字段，UserName + Host 唯一标识一个用户，IP形式，IP段以%结尾；支持填入%；为空默认等于%）。
Description String 是 用户备注信息
CreateTime Timestamp 是 创建时间
UpdateTime Timestamp 是 最后更新时间
ReadOnly Integer 是 只读标记，0：否， 1：该账号的sql请求优先选择备机执行，备机不可用时选择主机执行，2：优先选择备机执行，备机不可用时操作失败。
