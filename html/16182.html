1. 接口描述

本接口（DescribeBackupTime）用于获取云数据库的备份时间。后台系统将根据此配置定期进行实例备份。

2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见公共请求参数。

参数名称 是否必选 类型 描述
Action 是 String 公共参数，本接口取值：DescribeBackupTime。
Version 是 String 公共参数，本接口取值：2017-03-12。
InstanceIds.N 是 Array of String 实例ID，形如：tdsql-ow728lmc，可以通过 DescribeDBInstances 查询实例详情获得。

3. 输出参数

参数名称 类型 描述
TotalCount Integer 返回的配置数量
Items DBBackupTimeConfig 实例备份时间配置信息
RequestId String 唯一请求ID，每次请求都会返回。定位问题时需要提供该次请求的RequestId。

4. 错误码

错误码 描述
FailedOperation.OssOperationFailed 请求后端接口失败。
InternalError.DbOperationFailed 查询数据库失败。
InternalError.GetInstanceDetailFailed 获取实例详情失败。
InternalError.InnerSystemError 内部系统错误。

5. 示例

示例1 查询云数据库实例的备份时间窗口配置

请求参数

https://mariadb.tencentcloudapi.com/?Action=DescribeBackupTime
&InstanceIds.0=tdsql-fdpjf5zh
&<公共请求参数>
