1. 接口描述

本接口（ModifyDBInstancesProject）用于修改云数据库实例所属项目。

2. 输入参数

以下请求参数列表仅列出了接口请求参数和部分公共参数，完整公共参数列表见公共请求参数。

参数名称 是否必选 类型 描述
Action 是 String 公共参数，本接口取值：ModifyDBInstancesProject。
Version 是 String 公共参数，本接口取值：2017-03-12。
InstanceIds.N 是 Array of String 待修改的实例ID列表。实例 ID 形如：tdsql-ow728lmc，可以通过 DescribeDBInstances 查询实例详情获得。
ProjectId 是 Integer 要分配的项目 ID，可以通过 DescribeProjects 查询项目列表接口获取。

3. 输出参数

参数名称 类型 描述
RequestId String 唯一请求ID，每次请求都会返回。定位问题时需要提供该次请求的RequestId。

4. 错误码

错误码 描述
InternalError.DbOperationFailed 查询数据库失败。

5. 示例

示例1 修改云数据库实例所属项目

请求参数

https://mariadb.tencentcloudapi.com/?Action=ModifyDBInstancesProject
&CdbInstanceIds.0=tdsql-fdpjf5zh
&ProjectId=0
&<公共请求参数>

返回参数

{
  "Response": {
    "RequestId": "eac6b301-a322-493a-8e36-83b295459397"
  }
}
