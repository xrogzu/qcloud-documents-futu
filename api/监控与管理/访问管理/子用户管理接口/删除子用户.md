## 接口描述

本接口（DeleteUser ）可用于删除子用户。

请求域名：

```
cam.api.qcloud.com
```

## 输入参数

以下请求参数列表仅列出了接口请求参数，未列出 [公共请求参数](/document/product/248/4478)。

| 参数名称 | 类型   | 必选 | 描述         |
| -------- | ------ | ---- | ------------ |
| name     | string | 是   | 子用户用户名 |

## 输出参数

无。

http://cam.api.qcloud.com/v2/index.php

&SignatureMethod=HmacSHA256
