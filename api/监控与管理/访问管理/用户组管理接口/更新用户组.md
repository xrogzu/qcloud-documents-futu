## 接口描述

本接口（UpdateGroup）可用于更新用户组。

请求域名:

```
cam.api.qcloud.com
```

## 输入参数

以下请求参数列表仅列出了接口请求参数，未列出 [公共请求参数](/document/product/248/4478)。

| 参数名称  | 类型   | 必选 | 描述       |
| --------- | ------ | ---- | ---------- |
| groupId   | int    | 是   | 用户组 id  |
| groupName | string | 否   | 用户组名   |
| remark    | string | 否   | 用户组描述 |

## 输出参数
无。
